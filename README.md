# Styunlen-s-Minecraft-Server-Status-GUI
我的MC私人服务器的状态获取器

> <font size=24>
网课末期通过端口映射开了一个小私服，然而自己家电脑开服会带来开服时间不确定的问题<br><br>
作为一个开发者，我想到了可以用程序来帮助我快速通知我的小伙伴们，我是否已经开服，这样我就不用每次开服都去通知他们啦(还是懒QWQ)<br><br>
通过MC服务器的通讯协议可以帮助我快速获取MC服务器的当前状态<br><br>
